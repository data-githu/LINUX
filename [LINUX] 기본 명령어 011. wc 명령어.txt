11. wc 명령어
파일 안에 단어의 개수 또는 라인수를 출력하는 명령어

예제 : 
# wc -l 명령어
# wc -l jobs.txt

옵션   
-l  : 라인수 
-w : 단어의 갯수 
-c :  철자의개수	

문제30.  집으로 가서(/root)  확장자가 .txt 인 파일을 조회하시오.
 #  cd
 #  ls -l  *.txt

문제31. 위에 출력되는 결과의 라인수를 출력하시오!
# ls -l *.txt >> list3.txt
# wc -l list2.txt

문제32. 위의 작업을 파일을 생성하지 않고 더 간단하게 하는 방법은?
# ls -l *.txt | wc-l

☞ pipe 명령어? 
앞의 명령어 | 뒤의 명령어 
앞의 명령어의 출력을 뒤의 명령어의 입력으로 보냄으로써 명령어의 실행결과를 다음 명령어로 전달하는 기능

문제33. /root 밑에 디렉토리(폴더) 가 몇 개 있는지 조회하시오!
# cd
# pwd
# ls -ld */ | wc -l

※ 설명 :  ls -ld  */  라고 하면  현재 디렉토리에 있는  디렉토리 리스트들을 보여달라는것입니다.  이 출력결과를 파이프 다음 명령어의 입력으로 전달 합니다.  파이프 다음 명령어가 wc -l 의 입력으로 전달하면 그 결과의 라인수를 출력합니다.
